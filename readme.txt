CsvExlConverterのご使用方法

CsvExlConverter.exe(コンパイル済ファイル)をダウンロードして、
適当な場所に設置してください。

Excelで、Expressに作成したエンティティ(テーブル)と同じハンドル名の
フィールド列を1列目に作成してください。

2列目以降に、データを入力していきます。

Excelファイルを保存してください。

CsvExlConverter.exeを起動します。
クマのアイコンに、コンバートするエクセルファイルをドラッグアンドドロップしてください。
エンティティ名に、Expressで作成したエンティティ名を入力します。

「XMLコンバート開始」ボタンをクリックします。

処理完了のメッセージが表示されますと、exeと同じ場所にXMLファイルが作成されております
ので、これをマイグレーションツールでExpressにインポートしてください。
